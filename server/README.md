# express-starterkit

Starter kit for express with autoreload.

# Modules required:

- express
- babel-cli
- babel-preset-es2015
- nodemon


## Installation:

```bash
# Dependencies:
npm install --save express
```
```bash
# Dependencies development:
npm install --save-dev babel-cli babel-preset-es2015 nodemon
```

## Compatibility ES6:
Create a .babelrc file at the root with {"presets": ["es2015"]}
